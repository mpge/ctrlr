# Ctrlr examples

Drop-in starter configs you can copy into your own project.

| Folder              | What it gives you                                                      |
| ------------------- | ---------------------------------------------------------------------- |
| `claude-quartet/`   | Four Claude Code panes (Backend / Frontend / Tests / Docs). A/B/X/Y send `1\n`–`4\n` to the focused pane — perfect for Claude's numbered prompts. |
| `dev-server-trio/`  | API, Web, and DB containers as panes. Face buttons jump straight to a pane; BACK stops them all. |

To use one:

```bash
cp examples/claude-quartet/* .
ctrlr start
```
