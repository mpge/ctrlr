# Security policy

## Supported versions

Until the project hits 1.0, security fixes target the `main` branch only.

## Reporting a vulnerability

Email **security@matthewpg.com** with:

- A description of the issue and its impact.
- Steps to reproduce.
- Your suggested remediation, if any.

We aim to acknowledge reports within 72 hours and to ship a fix or mitigation within 14 days for high-severity issues. Please do not open a public issue for security reports.
